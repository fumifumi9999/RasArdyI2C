インクルードパスの設定：
PJ階層→librariesにzipを解凍したフォルダをコピー
プロジェクト・エクスプローラからPJを右クリック→C/C++一般→パスおよびシンボル→インクルードタブ内でCとC++コンパイラ両方に対してPJ階層→libraries→追加したパッケージフォルダを追加
eclipse再起動→コンパイル→これが通ればOK

上記でもダメなら、
プロジェクト・エクスプローラからPJを右クリック→C/C++設定→設定→ツール設定タブ内でarduino Cとarduino C++コンパイラ両方に対してPJ階層→libraries→追加したパッケージフォルダを追加
eclipse再起動→これでコンパイル通るでしょう
